# Chip-8
Chip-8 emulator written in C++ and is a current work in progress.
## About Chip-8
Chip-8 is an interpreted programming language from the 1970s, originally designed to make games easier to write for 8-bit microcomputers. 
Rather than emulating real hardware, this project implements a virtual machine that interprets Chip-8 instructions directly — the same approach used by the original interpreters.
